# 简易记账软件

一个功能完整的个人记账网页应用，支持用户认证、收支记录、预算管理、数据可视化和数据导出等功能。

## 功能特性

### 核心功能
- ✅ **用户认证**：注册、登录、登出
- ✅ **交易记录**：添加、编辑、删除收支记录
- ✅ **分类管理**：支持多种预设分类和自定义分类
- ✅ **预算管理**：设置月预算并实时监控预算使用情况
- ✅ **数据可视化**：通过图表直观展示收支情况
- ✅ **数据导出**：支持导出为 CSV 和 Excel 格式
- ✅ **数据备份**：一键备份和恢复数据

### 技术亮点
- 💡 前后端分离架构
- 🔒 安全的密码哈希和会话管理
- 📊 多种图表类型展示数据
- 📱 响应式设计，适配多种设备
- 📄 详细的代码注释和架构文档

## 重要提示
本项目为动态网站，包含 Node.js 后端服务器和 SQLite 数据库。由于 GitHub Pages 仅支持静态网站托管，无法直接通过 GitHub Pages 运行此项目。如需使用本应用，请按照下方"快速开始"部分的说明，将代码下载到本地后运行。

## 快速开始

### 安装依赖
```bash
npm install
```

### 启动服务器
```bash
node server.js
```

### 访问应用

打开浏览器访问 `http://localhost:3000`

## 使用指南

### 1. 用户认证

#### 注册
1. 点击右上角的设置图标
2. 选择注册标签
3. 填写用户名和密码
4. 点击注册按钮

#### 登录
1. 点击右上角的设置图标
2. 选择登录标签
3. 输入用户名和密码
4. 点击登录按钮

### 2. 记录交易

#### 添加支出
1. 点击支出按钮
2. 选择合适的分类
3. 输入金额
4. 选择日期
5. 填写备注（可选）
6. 点击保存按钮

#### 添加收入
1. 点击收入按钮
2. 选择合适的分类
3. 输入金额
4. 选择日期
5. 填写备注（可选）
6. 点击保存按钮

### 3. 管理预算

1. 点击预算设置区域
2. 输入月预算金额
3. 点击保存按钮
4. 系统会实时显示预算使用情况和剩余金额

### 4. 查看报表

1. 系统会自动生成收支趋势图
2. 可以通过时间筛选器查看不同时期的收支情况
3. 点击查看全部按钮可以查看完整的交易记录

### 5. 导出数据

1. 点击导出按钮
2. 选择导出格式（CSV 或 Excel）
3. 系统会自动生成并下载文件

### 6. 备份与恢复

#### 备份数据
1. 点击备份按钮
2. 系统会生成包含所有数据的 JSON 文件
3. 保存该文件到安全位置

#### 恢复数据
1. 点击恢复按钮
2. 选择之前备份的 JSON 文件
3. 系统会将数据恢复到备份状态

## 安全说明

1. 密码使用 bcrypt 进行哈希存储
2. 会话信息存储在安全的 HTTP-only cookie 中
3. 所有 API 请求都经过认证和授权
4. 启用了请求频率限制，防止暴力攻击
5. 使用 helmet 设置了安全头部
